Don't have a FlightRadar24 sharing key?

Run this command on your system to get a sharing key:

docker run -it --rm ghcr.io/sdr-enthusiasts/docker-baseimage:qemu bash -c "$(curl -sSL https://raw.githubusercontent.com/sdr-enthusiasts/docker-flightradar24/main/get_adsb_key.sh)"

Quick answers for the signup process:

  • Email: Your FlightRadar24 account email
  • Sharing key: Leave blank (you'll get a new one)
  • MLAT: Answer "no"
  • Receiver type: Answer "4" (Network)
  • Connection type: Answer "1"
  • Host: 127.0.0.1
  • Port: 30005

After signup, you'll receive a sharing key. Copy it and paste below.
